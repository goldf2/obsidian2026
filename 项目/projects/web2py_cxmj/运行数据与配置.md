## 待检查

- DAL 是否通过 `folder=` 指向 `/data/<web2py_service>/<app>/databases`。
- 上传字段是否通过 `uploadfolder=` 或 `UploadField` 指向 `/data/<web2py_service>/<app>/uploads`。
- 真实配置是否位于 `/data/<web2py_service>/<app>/config/appconfig.ini`。
- app 内 `databases/`、`uploads/` 是否仍产生新文件。
- `sessions/errors/cache/logs` 是否进入代码目录。

## 排查命令

```bash
find /opt/<web2py_service>/applications -maxdepth 2 \( -name databases -o -name uploads -o -name sessions -o -name errors -o -name cache \) -ls
rg -n "folder=|uploadfolder|Field\\(.*['\"]upload['\"]|request\\.folder.*uploads" /opt/<web2py_service>/applications -g '*.py'
```
