# 统一导航框架设计方案

## 1. 概述

### 1.1 背景
当前系统存在多个独立应用（service_center, jsll001, hdpf, cx1 等），各应用有独立的导航系统，导致：
- 导航风格不统一
- 维护成本高
- 用户体验碎片化

### 1.2 目标
构建统一的页面框架，实现：
- 固定顶部导航 + 面包屑导航
- 子页面通过 iframe 嵌入，无需维护导航
- 菜单数据库化管理，支持多级菜单
- URL 同步，支持页面分享和收藏

```
│  ┌─────────────────────────────────────────────────────────┐ │
│  │                                                         │ │

```

### 2.1 组件说明

| 组件 | 功能 | 位置 |
|------|------|------|
| 顶部导航栏 | 显示一级菜单、用户信息、快捷操作 | 固定在页面顶部 |
| 面包屑导航 | 显示当前页面路径，支持返回上级 | 导航栏下方 |
| iframe 容器 | 加载子应用页面内容 | 主内容区域 |
| 菜单管理 | 数据库存储菜单，支持动态配置 | 后台管理 |

## 3. 数据库设计

### 3.1 菜单表 (menu_items)

```sql
CREATE TABLE menu_items (
    id              INTEGER PRIMARY KEY,
    menu_group      VARCHAR(50) DEFAULT 'main',  -- 菜单组: main/sidebar
    title           VARCHAR(100) NOT NULL,        -- 菜单标题
    url             VARCHAR(500) DEFAULT '#',     -- 链接地址
    icon            VARCHAR(50),                  -- 图标 (Font Awesome)
    parent_id       INTEGER REFERENCES menu_items(id), -- 父级菜单
    sort_order      INTEGER DEFAULT 0,            -- 排序
    is_active       BOOLEAN DEFAULT TRUE,         -- 是否启用
    open_in_iframe  BOOLEAN DEFAULT TRUE,         -- 是否在iframe中打开
    target_app      VARCHAR(100),                 -- 目标应用名
    created_on      DATETIME,
    modified_on     DATETIME
);
```

### 3.2 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| menu_group | 菜单分组 | `main`=顶部导航, `quick`=快捷入口 |
| url | 页面地址 | `/jsll001/default/index` |
| icon | Font Awesome 图标 | `fa-home`, `fa-cog` |
| parent_id | 父菜单ID，NULL表示一级菜单 | `1`, `NULL` |
| open_in_iframe | 是否在iframe中打开 | `TRUE`=iframe, `FALSE`=新窗口 |
| target_app | 目标应用 | `jsll001`, `hdpf` |

## 5. 父子页面通信

### 5.1 通信机制

使用 `postMessage` API 实现父子页面双向通信：

```
┌──────────────────┐         postMessage         ┌──────────────────┐
│                  │  ─────────────────────────> │                  │
│   主框架 (Parent) │                             │  子页面 (iframe)  │
│                  │  <───────────────────────── │                  │
└──────────────────┘         postMessage         └──────────────────┘
```

### 5.2 消息类型

| 消息类型 | 方向 | 说明 |
|----------|------|------|
| `PAGE_READY` | 子→父 | 子页面加载完成 |
| `PAGE_INFO` | 子→父 | 子页面信息（标题、路径） |
| `NAVIGATE` | 父→子 | 通知子页面导航 |
| `UPDATE_BREADCRUMB` | 子→父 | 更新面包屑 |

### 5.3 主框架代码

```javascript
// frame-controller.js

class FrameController {
    constructor() {
        this.iframe = document.getElementById('content-frame');
        this.breadcrumb = document.getElementById('breadcrumb');
        this.currentPath = '';

        this.init();
    }

    init() {
        // 监听子页面消息
        window.addEventListener('message', (e) => this.handleMessage(e));

        // 监听菜单点击
        document.querySelectorAll('[data-frame-url]').forEach(link => {
            link.addEventListener('click', (e) => {
                e.preventDefault();
                this.navigate(link.dataset.frameUrl, link.dataset.title);
            });
        });

        // 监听浏览器前进/后退
        window.addEventListener('popstate', (e) => {
            if (e.state && e.state.url) {
                this.loadPage(e.state.url, false);
            }
        });
    }

    handleMessage(event) {
        const data = event.data;

        switch (data.type) {
            case 'PAGE_READY':
                console.log('子页面加载完成');
                break;

            case 'PAGE_INFO':
                this.updateBreadcrumb(data.breadcrumb);
                this.updateUrl(data.path);
                break;

            case 'NAVIGATE':
                this.navigate(data.url, data.title);
                break;
        }
    }

    navigate(url, title) {
        this.loadPage(url, true);
        if (title) {
            document.title = title + ' - 创新数智中心';
        }
    }

    loadPage(url, pushState = true) {
        this.iframe.src = url;
        this.currentPath = url;

        if (pushState) {
            const stateUrl = '/service_center/default/index#' + encodeURIComponent(url);
            history.pushState({ url: url }, '', stateUrl);
        }
    }

    updateBreadcrumb(items) {
        if (!items || items.length === 0) return;

        let html = '<li class="breadcrumb-item"><a href="#" data-frame-url="/">首页</a></li>';

        items.forEach((item, index) => {
            if (index === items.length - 1) {
                html += `<li class="breadcrumb-item active">${item.title}</li>`;
            } else {
                html += `<li class="breadcrumb-item"><a href="#" data-frame-url="${item.url}">${item.title}</a></li>`;
            }
        });

        this.breadcrumb.innerHTML = html;
    }

    updateUrl(path) {
        const newUrl = '/service_center/default/index#' + encodeURIComponent(path);
        history.replaceState({ url: path }, '', newUrl);
    }
}

// 初始化
document.addEventListener('DOMContentLoaded', () => {
    window.frameController = new FrameController();
});
```

### 5.4 子页面代码

```javascript
// child-frame.js (子页面引入)

(function() {
    // 检测是否在 iframe 中
    const isInIframe = window.self !== window.top;

    if (isInIframe) {
        // 隐藏子页面的导航栏
        document.addEventListener('DOMContentLoaded', function() {
            const navbar = document.querySelector('.navbar');
            if (navbar) navbar.style.display = 'none';

            const sidebar = document.querySelector('.sidebar-drawer');
            if (sidebar) sidebar.style.display = 'none';

            const menuBtn = document.querySelector('.menu-toggle-btn');
            if (menuBtn) menuBtn.style.display = 'none';

            // 调整内容区域
            const content = document.querySelector('.main-content, .container-fluid');
            if (content) {
                content.style.marginTop = '0';
                content.style.paddingTop = '20px';
            }
        });

        // 通知父页面已加载
        window.parent.postMessage({
            type: 'PAGE_READY'
        }, '*');

        // 发送页面信息
        window.parent.postMessage({
            type: 'PAGE_INFO',
            path: window.location.pathname,
            title: document.title,
            breadcrumb: window.pageBreadcrumb || []
        }, '*');
    }
})();
```

### 6.2 前端渲染

```javascript
function renderMenu(items, container) {
    let html = '';

    items.forEach(item => {
        if (item.children && item.children.length > 0) {
            // 有子菜单
            html += `
                <li class="nav-item dropdown">
                    <a class="nav-link dropdown-toggle" href="#" data-toggle="dropdown">
                        ${item.icon ? '<i class="fa ' + item.icon + '"></i> ' : ''}
                        ${item.title}
                    </a>
                    <div class="dropdown-menu">
                        ${item.children.map(child => `
                            <a class="dropdown-item" href="#"
                               data-frame-url="${child.url}"
                               data-title="${child.title}">
                                ${child.icon ? '<i class="fa ' + child.icon + '"></i> ' : ''}
                                ${child.title}
                            </a>
                        `).join('')}
                    </div>
                </li>
            `;
        } else {
            // 无子菜单
            html += `
                <li class="nav-item">
                    <a class="nav-link" href="#"
                       data-frame-url="${item.url}"
                       data-title="${item.title}">
                        ${item.icon ? '<i class="fa ' + item.icon + '"></i> ' : ''}
                        ${item.title}
                    </a>
                </li>
            `;
        }
    });

    container.innerHTML = html;
}
```

## 9. 注意事项

### 9.1 安全性
- iframe 应限制 `sandbox` 属性
- postMessage 应验证来源 `origin`
- 敏感操作需要 CSRF 保护

### 9.2 兼容性
- 子页面需要引入 `child-frame.js`
- 老页面需要逐步迁移适配

### 9.3 性能
- iframe 预加载常用页面
- 菜单数据可缓存到 localStorage
