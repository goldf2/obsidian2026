## 1. 概述

### 1.1 项目背景

在 service_center 应用中集成阿里云短信服务，用于：
- 批量导入用户后发送初始密码
- 用户密码重置通知
- 系统通知（可选）
- 登录验证码（可选）

### 1.2 技术选型

| 项目 | 选择 | 说明 |
|------|------|------|
| 短信服务商 | 阿里云短信 | 稳定、国内主流 |
| API版本 | 2017-05-25 | 阿里云短信API最新版 |
| 集成方式 | 自封装HTTP调用 | 不依赖官方SDK，减少依赖 |

### 1.3 架构位置

```
┌─────────────────────────────────────────────────────────────┐
│                      service_center                          │
├─────────────────────────────────────────────────────────────┤
│  用户管理 (mgr)  │  短信服务 (sms)  │  菜单管理 (menu)       │
```

## 2. 前置准备（阿里云配置）

### 2.1 开通短信服务

1. 登录阿里云控制台：https://dysms.console.aliyun.com/
2. 开通短信服务（首次需要）
3. 完成企业实名认证（发送营销类短信需要）

### 2.2 创建 AccessKey

**推荐使用 RAM 子用户，避免使用主账号 AK**

1. 访问 RAM 控制台：https://ram.console.aliyun.com/
2. 创建用户：
   - 用户名：`sms-service`
   - 访问方式：勾选「OpenAPI 调用访问」
3. 保存 AccessKey ID 和 AccessKey Secret
4. 授权策略：`AliyunDysmsFullAccess`

### 2.3 申请短信签名

1. 进入短信控制台 → 国内消息 → 签名管理
2. 添加签名：
   - 签名名称：如 `创新模具`
   - 签名来源：企事业单位名称
   - 上传证明：营业执照
3. 等待审核（1-2个工作日）

### 2.4 申请短信模板

**模板1：初始密码通知**
```
模板名称：用户初始密码通知
模板内容：您的${system}账号已创建，用户名：${username}，初始密码：${password}，请尽快登录修改密码。
变量属性：
  - system: 字符串，最长10字
  - username: 字符串，最长20字
  - password: 字符串，最长20字
```

**模板2：密码重置通知（可选）**
```
模板名称：密码重置通知
模板内容：您的${system}账号密码已重置，新密码：${password}，请尽快登录修改。
```

**模板3：验证码（可选）**
```
模板名称：登录验证码
模板内容：您的验证码是${code}，5分钟内有效，请勿泄露给他人。
```

### 4.1 阿里云短信模块

**文件**: `modules/aliyun_sms.py`

```python
# -*- coding: utf-8 -*-
"""
阿里云短信服务封装模块

使用方法:
    from applications.service_center.modules.aliyun_sms import AliyunSMS

    sms = AliyunSMS(
        access_key_id='your_ak',
        access_key_secret='your_sk',
        sign_name='创新模具'
    )

    result = sms.send(
        phone='13800138000',
        template_code='SMS_123456789',
        template_params={'username': 'zhangsan', 'password': 'abc123'}
    )
"""

import hmac
import hashlib
import base64
import uuid
import json
import urllib.parse
import urllib.request
from datetime import datetime


class AliyunSMS:
    """阿里云短信服务类"""

    API_VERSION = '2017-05-25'
    ENDPOINT = 'dysmsapi.aliyuncs.com'
    REGION_ID = 'cn-hangzhou'

    def __init__(self, access_key_id, access_key_secret, sign_name,
                 endpoint=None, region_id=None):
        """
        初始化短信服务

        参数:
            access_key_id: 阿里云 AccessKey ID
            access_key_secret: 阿里云 AccessKey Secret
            sign_name: 短信签名（需在阿里云控制台申请）
            endpoint: API端点（可选）
            region_id: 区域ID（可选）
        """
        self.access_key_id = access_key_id
        self.access_key_secret = access_key_secret
        self.sign_name = sign_name
        self.endpoint = endpoint or self.ENDPOINT
        self.region_id = region_id or self.REGION_ID

    def send(self, phone, template_code, template_params=None):
        """
        发送短信

        参数:
            phone: 手机号码（支持多个，用逗号分隔，最多1000个）
            template_code: 短信模板CODE
            template_params: 模板参数字典

        返回:
            {
                'success': True/False,
                'code': '响应码',
                'message': '响应消息',
                'biz_id': '业务ID',
                'request_id': '请求ID'
            }
        """
        params = {
            'Action': 'SendSms',
            'PhoneNumbers': phone,
            'SignName': self.sign_name,
            'TemplateCode': template_code,
        }

        if template_params:
            params['TemplateParam'] = json.dumps(template_params, ensure_ascii=False)

        try:
            response = self._request(params)
            return {
                'success': response.get('Code') == 'OK',
                'code': response.get('Code'),
                'message': response.get('Message'),
                'biz_id': response.get('BizId'),
                'request_id': response.get('RequestId')
            }
        except Exception as e:
            return {
                'success': False,
                'code': 'ClientError',
                'message': str(e),
                'biz_id': None,
                'request_id': None
            }

    def query_send_details(self, phone, biz_id=None, send_date=None, page_size=10, current_page=1):
        """
        查询发送详情

        参数:
            phone: 手机号码
            biz_id: 业务ID（可选）
            send_date: 发送日期，格式 yyyyMMdd（可选，默认今天）
            page_size: 每页数量
            current_page: 当前页码

        返回:
            发送详情列表
        """
        if send_date is None:
            send_date = datetime.now().strftime('%Y%m%d')

        params = {
            'Action': 'QuerySendDetails',
            'PhoneNumber': phone,
            'SendDate': send_date,
            'PageSize': str(page_size),
            'CurrentPage': str(current_page),
        }

        if biz_id:
            params['BizId'] = biz_id

        try:
            response = self._request(params)
            return {
                'success': response.get('Code') == 'OK',
                'total': response.get('TotalCount', 0),
                'details': response.get('SmsSendDetailDTOs', {}).get('SmsSendDetailDTO', [])
            }
        except Exception as e:
            return {
                'success': False,
                'message': str(e),
                'total': 0,
                'details': []
            }

    def _request(self, params):
        """发送API请求"""
        # 公共参数
        common_params = {
            'Format': 'JSON',
            'Version': self.API_VERSION,
            'AccessKeyId': self.access_key_id,
            'SignatureMethod': 'HMAC-SHA1',
            'Timestamp': datetime.utcnow().strftime('%Y-%m-%dT%H:%M:%SZ'),
            'SignatureVersion': '1.0',
            'SignatureNonce': str(uuid.uuid4()),
            'RegionId': self.region_id,
        }

        # 合并参数
        all_params = {**common_params, **params}

        # 生成签名
        signature = self._sign(all_params)
        all_params['Signature'] = signature

        # 发送请求
        url = f'https://{self.endpoint}/?{urllib.parse.urlencode(all_params)}'

        req = urllib.request.Request(url)
        req.add_header('Content-Type', 'application/x-www-form-urlencoded')

        with urllib.request.urlopen(req, timeout=30) as resp:
            return json.loads(resp.read().decode('utf-8'))

    def _sign(self, params):
        """生成签名"""
        # 按参数名排序
        sorted_params = sorted(params.items(), key=lambda x: x[0])

        # URL编码
        def encode(s):
            return urllib.parse.quote(str(s), safe='~')

        # 构建待签名字符串
        query_string = '&'.join([f'{encode(k)}={encode(v)}' for k, v in sorted_params])
        string_to_sign = f'GET&%2F&{encode(query_string)}'

        # HMAC-SHA1 签名
        key = (self.access_key_secret + '&').encode('utf-8')
        signature = hmac.new(key, string_to_sign.encode('utf-8'), hashlib.sha1).digest()

        return base64.b64encode(signature).decode('utf-8')


class SMSService:
    """短信服务封装（支持从数据库读取配置）"""

    def __init__(self, db):
        """
        初始化短信服务

        参数:
            db: web2py DAL 数据库连接
        """
        self.db = db
        self._client = None
        self._config = None

    def _get_config(self):
        """获取启用的短信配置"""
        if self._config is None:
            self._config = self.db(self.db.sms_config.is_active == True).select().first()
        return self._config

    def _get_client(self):
        """获取短信客户端"""
        if self._client is None:
            config = self._get_config()
            if not config:
                raise ValueError('未配置短信服务')

            self._client = AliyunSMS(
                access_key_id=config.access_key_id,
                access_key_secret=config.access_key_secret,
                sign_name=config.sign_name,
                endpoint=config.endpoint,
                region_id=config.region_id
            )
        return self._client

    def get_template(self, code):
        """根据标识获取模板"""
        return self.db(
            (self.db.sms_template.code == code) &
            (self.db.sms_template.is_active == True)
        ).select().first()

    def send(self, phone, template_code, template_params=None, sent_by=None, batch_id=None):
        """
        发送短信并记录日志

        参数:
            phone: 手机号
            template_code: 系统模板标识 或 阿里云模板CODE
            template_params: 模板参数
            sent_by: 发送人ID
            batch_id: 批次ID

        返回:
            发送结果
        """
        config = self._get_config()
        if not config:
            return {'success': False, 'message': '未配置短信服务'}

        # 检查是否为系统模板标识
        template = self.get_template(template_code)
        aliyun_template_code = template.template_code if template else template_code

        # 发送短信
        client = self._get_client()
        result = client.send(phone, aliyun_template_code, template_params)

        # 记录日志
        self.db.sms_log.insert(
            phone=phone,
            template_code=aliyun_template_code,
            template_params=template_params,
            sign_name=config.sign_name,
            status='success' if result['success'] else 'failed',
            response_code=result.get('code'),
            response_msg=result.get('message'),
            biz_id=result.get('biz_id'),
            request_id=result.get('request_id'),
            sent_by=sent_by,
            batch_id=batch_id
        )
        self.db.commit()

        return result

    def send_batch(self, recipients, template_code, sent_by=None):
        """
        批量发送短信

        参数:
            recipients: 收件人列表 [{'phone': '138...', 'params': {...}}, ...]
            template_code: 模板标识
            sent_by: 发送人ID

        返回:
            批量发送结果
        """
        import uuid as uuid_module
        batch_id = str(uuid_module.uuid4())[:8]

        results = {
            'batch_id': batch_id,
            'total': len(recipients),
            'success': 0,
            'failed': 0,
            'details': []
        }

        for recipient in recipients:
            phone = recipient.get('phone')
            params = recipient.get('params', {})

            if not phone:
                results['failed'] += 1
                results['details'].append({
                    'phone': phone,
                    'success': False,
                    'message': '手机号为空'
                })
                continue

            result = self.send(
                phone=phone,
                template_code=template_code,
                template_params=params,
                sent_by=sent_by,
                batch_id=batch_id
            )

            if result['success']:
                results['success'] += 1
            else:
                results['failed'] += 1

            results['details'].append({
                'phone': phone,
                'success': result['success'],
                'message': result.get('message', '')
            })

        return results
```

```python
@auth.requires_login()
def user_batch_send_sms():
    """批量导入后发送短信通知"""
    _check_admin()

    data = json.loads(request.body.read() or '{}')
    users = data.get('users', [])

    if not users:
        return response.json({'error': '没有要发送的用户'})

    # 检查短信配置
    config = db(db.sms_config.is_active == True).select().first()
    if not config:
        return response.json({'error': '未配置短信服务，请先在短信管理中配置'})

    # 获取初始密码模板
    template = db(db.sms_template.code == 'password_init').select().first()
    if not template:
        return response.json({'error': '未配置初始密码通知模板'})

    # 构建收件人列表
    recipients = []
    skipped = []

    for user in users:
        phone = user.get('phone', '').strip()
        if not phone:
            skipped.append({'username': user.get('username'), 'reason': '无手机号'})
            continue

        recipients.append({
            'phone': phone,
            'params': {
                'system': '创新数字中心',
                'username': user.get('username'),
                'password': user.get('password')
            }
        })

    if not recipients:
        return response.json({'error': '没有可发送的用户（均无手机号）', 'skipped': skipped})

    # 批量发送
    from applications.service_center.modules.aliyun_sms import SMSService
    sms_service = SMSService(db)
    result = sms_service.send_batch(
        recipients=recipients,
        template_code='password_init',
        sent_by=auth.user_id
    )

    result['skipped'] = skipped
    return response.json(result)
```

## 8. 安全设计

### 8.1 凭证安全

| 风险 | 措施 |
|------|------|
| AK/SK 泄露 | 使用 RAM 子用户，最小权限原则 |
| 数据库泄露 | access_key_secret 字段使用 password 类型（加密存储） |
| 日志泄露 | 日志中不记录完整密码，仅记录脱敏手机号 |

### 8.2 发送安全

| 风险 | 措施 |
|------|------|
| 短信轰炸 | 同一手机号每分钟最多发送1条 |
| 费用失控 | 设置每日发送上限 (daily_limit) |
| 未授权发送 | 仅管理员可操作发送功能 |

### 8.3 频率限制实现

```python
def _check_rate_limit(phone, interval_seconds=60):
    """检查发送频率限制"""
    from datetime import datetime, timedelta

    cutoff = datetime.now() - timedelta(seconds=interval_seconds)
    recent = db(
        (db.sms_log.phone == phone) &
        (db.sms_log.sent_on > cutoff)
    ).count()

    return recent == 0  # True 表示可以发送
```

| 错误码 | 说明 | 处理建议 |

| isv.BUSINESS_LIMIT_CONTROL | 触发限流 | 降低发送频率 |
| isv.AMOUNT_NOT_ENOUGH | 余额不足 | 充值 |
